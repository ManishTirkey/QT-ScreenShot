Introduction

Catch and Hold is a powerful screenshot tool that allows you to capture, edit, and save screenshots with ease. Whether you need to capture a specific area of your screen, save it for later, or share it with others, Catch and Hold provides a simple and intuitive interface to accomplish these tasks.

Installation

To install Catch and Hold:

  1. Download the installer from our website
  2. Run the installer and follow the on-screen instructions
  3. Choose whether to start the application automatically with Windows
  4. Complete the installation

Getting Started

After installation, Catch and Hold will run in the system tray. You can access it by clicking on the icon in the system tray (usually located in the bottom-right corner of your screen).

Right-clicking on the tray icon will open a menu with various options:

  • Capture Screenshot: Start capturing a screenshot
  • Keyboard Shortcuts: View available keyboard shortcuts
  • Settings: Configure application settings
  • About: View information about the application
  • Quit: Exit the application

Taking Screenshots

To take a screenshot:

  1. Press the Print Screen key or click on the tray icon
  2. Your screen will darken, indicating that you're in capture mode
  3. Click and drag to select the area you want to capture
  4. Release the mouse button to capture the selected area
  5. The captured image will appear in a preview window

While selecting an area, you'll notice:

  • Green dotted lines following your cursor
  • A magnifying glass to help with precise selection
  • The dimensions (width Ã— height) of the selected area
Tip: For the most precise selection, use the magnifying glass to ensure you're capturing exactly what you need.

Editing Screenshots

After capturing a screenshot, you can:

  • Press Ctrl+S to save the screenshot
  • Press Ctrl+Shift+S to copy the screenshot to the clipboard

Keyboard Shortcuts

Catch and Hold provides several keyboard shortcuts to enhance your productivity:

Shortcut Action
Print Screen Capture screenshot (works system-wide)
ESC Cancel capture
Ctrl+S Save screenshot (in preview window)
Ctrl+Shift+S Copy to clipboard (in preview window)

Troubleshooting

If you encounter any issues with Catch and Hold, try these solutions:

  • The application doesn't start:
    • Check if the application is already running in the system tray
    • Try running the application as administrator
    • Reinstall the application
  • Print Screen key doesn't work:
    • Some laptops require pressing Fn+Print Screen
    • Check if another application is using the Print Screen key
    • Try restarting the application
  • Can't save screenshots:
    • Make sure you have write permissions to the destination folder
    • Try saving to a different location

If you continue to experience issues, please contact our support team at support@yourcompany.com.
